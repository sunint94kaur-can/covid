# COVID-19 Data Analysis using SQL

## Overview
This project analyzes COVID-19 data using SQL Server to understand:
- Death percentage by country
- Infection rate compared to population
- Countries with highest infection counts
- Vaccination progress using rolling totals

## Skills Demonstrated
- SQL aggregation (MAX, SUM)
- Data type conversion (CAST)
- Window functions (SUM OVER)
- CTEs
- Percentage calculations
- Real-world data handling

## Tools
- SQL Server
- SQL Server Management Studio (SSMS)

## Dataset
COVID-19 dataset (CovidDeaths & CovidVaccinations tables)
